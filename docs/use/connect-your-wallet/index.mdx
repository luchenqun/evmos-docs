# 钱包概述

Evmos网络支持Cosmos和ERC-20代币。有几个钱包与该网络兼容。其中一些钱包支持Ledger连接。请参考下表获取更多信息。我们推荐使用Metamask和Keplr钱包。这两个钱包也支持与Ledger的连接。有关Ledger的更多信息，请前往[此处](./connect-your-wallet/ledger)。

| 钱包                 | 支持  | Ledger | 详情                                  |
| ---------------------- | ---------- |---------|-------------------------------------------
| [Metamask](https://metamask.io/)               | ✅   | ✅    | 移动端和浏览器。支持ERC-20代币，但不支持Cosmos代币|
| [Keplr](https://www.keplr.app/)                  | ✅ | ✅        | 移动端和浏览器。支持Cosmos代币，但即将支持ERC-20代币 |
| [WalletConnect v2](https://docs.walletconnect.com/2.0)       | ✅   | ✅      | 仅支持[Dashboard Assets](https://app.evmos.org/assets)
| [WalletConnect v1](https://docs.walletconnect.com/1.0/)       | 🟡   | ✅      | 仅支持[Dashboard](https://app.evmos.org/)（不包括Assets）。即将停用。 |
| [Ledger](https://www.ledger.com/)                 | ✅   | ✅      | 桌面端和移动端。支持Metamask、Keplr和CLI |
| [Trust Wallet](https://trustwallet.com/)           | ✅   | ❌      | 移动端和浏览器 |
| [Leap Wallet](https://www.leapwallet.io/)           | ✅   | ❌      | 浏览器 |
| [Coinbase Wallet](https://www.coinbase.com/wallet)           | ❌   | ✅      | 不支持 |
| [Cosmostation](https://wallet.cosmostation.io/cosmos)           | ✅   | ❌      | 支持浏览器和移动端 |
| [Cypher](https://www.cypherwallet.io/)           | ✅   | ❌      | 支持移动端 |

# Wallet Overview

Evmos network supports both Cosmos and ERC-20 coins. There are several wallets that work with the network. Some of these wallets support Ledger connectivity. Consult the table below for more information. We recommend Metamask and Keplr wallets. These two wallets also work with the Ledger. For more information about Ledger, proceed to [here](./connect-your-wallet/ledger).

| Wallet                 | Supported  | Ledger | Detail                                  |
| ---------------------- | ---------- |---------|-------------------------------------------
| [Metamask](https://metamask.io/)               | ✅   | ✅    | Mobile & Browsers. Supports ERC-20 tokens, but no Cosmos coins|
| [Keplr](https://www.keplr.app/)                  | ✅ | ✅        | Mobile & Browsers. Supports Cosmos coins but ERC-20 support coming soon |
| [WalletConnect v2](https://docs.walletconnect.com/2.0)       | ✅   | ✅      | Supported only on the [Dashboard Assets](https://app.evmos.org/assets)
| [WalletConnect v1](https://docs.walletconnect.com/1.0/)       | 🟡   | ✅      | Supported only on the [Dashboard](https://app.evmos.org/) (excluding Assets). Deprecating soon. |
| [Ledger](https://www.ledger.com/)                 | ✅   | ✅      | Desktop & Mobile. Supported on Metamask, Keplr, and CLI |
| [Trust Wallet](https://trustwallet.com/)           | ✅   | ❌      | Mobile & Browser |
| [Leap Wallet](https://www.leapwallet.io/)           | ✅   | ❌      | Brwoser |
| [Coinbase Wallet](https://www.coinbase.com/wallet)           | ❌   | ✅      | Not Supported |
| [Cosmostation](https://wallet.cosmostation.io/cosmos)           | ✅   | ❌      | Supported on Browswer and Mobile |
| [Cypher](https://www.cypherwallet.io/)           | ✅   | ❌      | Supported on Mobile |
